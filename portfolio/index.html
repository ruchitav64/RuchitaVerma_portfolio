Flowchart Plagiarism Detection

Project Case Study & System Design

Problem Statement

Detecting plagiarism in flowcharts is challenging due to variations in structure, layout, symbols, and drawing styles. Traditional text-based plagiarism detection methods fail to capture visual and structural similarities.

Objective

To design an intelligent system that detects plagiarism in flowcharts by analyzing structural and visual similarities using image processing and machine learning techniques.

Proposed Approach

  • Preprocess flowchart images using grayscale conversion and thresholding
  • Detect shapes such as process, decision, and terminator blocks
  • Extract relationships and convert flowcharts into graph representations
  • Compare graphs using similarity metrics

Tech Stack

Python, OpenCV, Image Processing, Machine Learning

Challenges

  • Handling different drawing styles and orientations
  • Shape detection under noisy conditions
  • Graph comparison complexity

Current Status

  • System design and algorithm planning have been completed.
  • A custom dataset of flowchart images has been successfully created.
  • Feature extraction and similarity analysis implementation is currently in progress.
